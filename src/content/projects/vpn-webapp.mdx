<Lang val="ru">
В этом проекте я выступил в роли Lead Developer и Архитектора. Задача: создать полностью автономный бизнес по продаже VPN.

Я не просто сверстал интерфейс, я построил систему целиком:

- **Архитектура:** Next.js (App Router) + Zustand для мгновенного отклика UI.
- **DevOps:** Настроил CI/CD пайплайны, автоматический деплой и управление конфигурациями серверов.
- **Биллинг:** Интеграция YooKassa/Ozon Bank со сложной логикой рекуррентных платежей и обработки ошибок.

Результат: система работает без участия человека, пользователь получает доступ через 5 секунд после оплаты.

</Lang>

<Lang val="en">
I acted as Lead Developer and Architect. The goal: create a fully autonomous VPN business.

I built the entire ecosystem:

- **Architecture:** Next.js (App Router) + Zustand for instant UI response.
- **DevOps:** CI/CD pipelines, auto-deployment, and server configuration management.
- **Billing:** Integration with YooKassa/Ozon Bank handling complex recurring payments.

The result is a zero-touch system where users get access 5 seconds after payment.
