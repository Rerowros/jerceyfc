Разработал архитектуру фронтенда для регионального портала услуг связи. Главный вызов — скорость и SEO для сотен регионов с разными тарифами. Использовал Astro SSR, что позволило генерировать уникальный HTML для каждого запроса на лету, сохраняя при этом производительность статики.

Внедрил кастомный ETL-пайплайн: скрипты на Python преобразуют сырые CSV-выгрузки тарифов в оптимизированные JSON-структуры. Для защиты API реализовал Rate Limiting и валидацию токенов. Чтобы не просаживать метрики Core Web Vitals, тяжелые скрипты аналитики и виджеты загружаются только после взаимодействия пользователя (паттерн Facade).
