I acted as Lead Developer and Architect. The goal: create a fully autonomous VPN business.

I built the entire ecosystem:

- **Architecture:** Next.js (App Router) + Zustand for instant UI response.
- **DevOps:** CI/CD pipelines, auto-deployment, and server configuration management.
- **Billing:** Integration with YooKassa/Ozon Bank handling complex recurring payments.

The result is a zero-touch system where users get access 5 seconds after payment.
