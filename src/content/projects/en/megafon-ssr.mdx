Architected the frontend for a major ISP portal, targeting maximum SEO performance across hundreds of locations. By leveraging Astro in SSR mode, I achieved dynamic HTML generation per region while maintaining static-site speed.

I built a custom ETL pipeline to handle complex pricing logic: scripts transform raw CSV tariff data into optimized JSON structures. The system is hardened with rate limiting and strict input validation. Additionally, I implemented an interaction-based 'Lazy Loading' strategy for third-party scripts, drastically reducing initial payload size.
